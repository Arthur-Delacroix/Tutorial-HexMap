## 关于本教程
这个教程原文是Jasper Flick大神的文章。整个教程包含了完成一套地图系统的方方面面，难度也是循序渐进。有很多代码和思想都非常值得借鉴。所以在征求了Jasper Flick大神的统一之后，决定将整套教程翻译出来，供大家学习参考。

教程一共分为两个部分，一部分是文字说明，发表在我的[Github个人博客](https://arthur-delacroix.github.io/)上。另一部分是代码，在我的公开仓库[Tutorial-HexMap](https://github.com/Arthur-Delacroix/Tutorial-HexMap)中。

文字部分参考了知乎用户[沈琰](https://www.zhihu.com/people/chen-yan-60-7)的文章。而我自己的博客文章，对Jasper Flick大神英文原版翻译之后，进行了一定的润色，和更多比较详细的说明，更有助于大家理解。

代码部分我通过理解Jasper Flick大神的思路，做了非常多的注释，这些注释说明了我对代码的理解思路。并且纠正了Jasper Flick大神文章和代码的部分疏漏。

我的Hex Map中文教程，只会发布在我的个人博客中，将来打算制作视频版，发布在B站，其余地方出现的，均没有得到本人和Jasper Flick大神授权与同意。希望大家对这个教程的中文版提出宝贵意见，共同学习，共同进步。
